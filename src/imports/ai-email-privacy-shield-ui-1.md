Design UI components for a Chrome extension called **AI Email Privacy Shield** that integrates with Gmail.

IMPORTANT:
Do NOT design a full webpage or Gmail interface.
Do NOT create a fake inbox layout.
Design only small UI components that overlay on top of Gmail.

Create the following components:

1. **Inbox Risk Badge Component**
   A small pill-shaped badge that appears next to Gmail email subjects.

Variants:
• SAFE (green)
• SUSPICIOUS (yellow)
• HIGH RISK (red)

Design requirements:
• minimal pill badge
• height around 20–24px
• rounded corners
• small icon + short label
• designed to fit inline with Gmail email rows

---

2. **Email Security Analysis Popup Card (Appears when an email is opened)**

When a user opens an email message in Gmail, the extension shows a **floating popup window card above the email content**.

IMPORTANT:
This must be a **floating popup card overlay**, not a webpage.

Card width: around 350–380px.

Content:
• Title: "AI Email Privacy Shield – Email Security Report"
• Sender Trust Score (progress bar)
• Email Privacy Score (progress bar)

Threat detection list:
• Tracking Pixel Detected
• Suspicious Links
• Phishing Language
• Dangerous Attachments

Buttons:
• View Details
• Block Trackers
• Report Phishing

Style:
• modern cybersecurity card
• rounded corners
• subtle shadow
• dark theme
• compact popup window style

---

3. **Chrome Extension Popup Window**

This appears when the user clicks the extension icon in Chrome.

Popup size:
380px width
500–600px height

Sections:
• Extension header with logo
• Current email status
• Sender information
• Trust Score
• Privacy Score
• Threat summary
• Buttons: Scan Email, View Full Analysis, Settings

Style:
Modern cybersecurity dashboard UI with green, yellow, and red risk indicators.

The design should look like **security software overlays and popup windows**, not a full webpage.

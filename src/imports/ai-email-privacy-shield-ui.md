Design a complete UI for a cybersecurity Chrome extension called **“AI Email Privacy Shield”** that integrates with Gmail to protect users from phishing, tracking pixels, and malicious links.

The UI should include three main components:

1. **Inbox Risk Indicator (Gmail Integration)**
   Design a small badge that appears next to each email subject in the Gmail inbox list.

Badge states:
• Safe (Green)
• Suspicious (Yellow)
• High Risk (Red)

Each badge should contain:
• risk icon
• short label (SAFE, SUSPICIOUS, HIGH RISK)

Style:
• pill-shaped label
• compact and minimal
• fits inside Gmail inbox rows without breaking layout

---

2. **Email Security Analysis Panel (Injected inside opened email)**

Design a modern security card that appears at the top of an opened email showing a privacy and threat analysis.

Panel contents:
• Header: "AI Email Privacy Shield – Email Security Report"
• Sender Trust Score (0–100 progress bar)
• Email Privacy Score (0–100 progress bar)

Threat detection section:
• Tracking Pixel Detected
• Suspicious Links Found
• Phishing Language Detected
• Dangerous Attachments

Each threat should show:
• icon
• short explanation
• severity color

Also include:
• “View Detailed Report” button
• “Block Trackers” button
• “Report Phishing” button

Panel style:
• floating card
• rounded corners
• subtle shadow
• width around 350–400px
• modern cybersecurity dashboard style

---

3. **Chrome Extension Popup Dashboard**

Design a popup window that appears when the user clicks the extension icon in Chrome.

Popup size:
380px width
500–600px height

Sections:
• Header with logo and extension name
• Current Email Status
• Sender information
• Trust Score indicator
• Privacy Score indicator
• Threat summary list

Threat summary examples:
• Hidden tracking pixel
• Suspicious domain
• Phishing keywords detected
• Unsafe link

Add buttons:
• Scan Email
• View Full Analysis
• Block Trackers
• Settings

---

Visual style:

Theme:
Modern cybersecurity dashboard

Color palette:
• Dark background
• Green = safe
• Yellow = warning
• Red = danger
• Blue = neutral UI elements

Typography:
Clean modern sans-serif

Icons:
Shield icons
Warning icons
Link icons
Tracker/eye icons

Design characteristics:
• minimal
• professional
• clean layout
• card-based UI
• clear security indicators
• user-friendly for non-technical users

Make the design suitable for a Chrome extension and Gmail overlay integration.
